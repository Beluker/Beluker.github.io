fiddly

An In-Depth Guide

with steps & a video

If you have never used inspector before, you might want to quickly go to the console and check if pasting is enabled, if not type allow pasting

The steps go as follow:
1open inspector
2navigate to the Sources panel
3look in the Assets folder, for the index.a9ff7d61.js file
4press ctrl + F to open the search bar
5paste Tp.sendBytePacket(kr(Hc[e(305)], Xi[e(285)])), into the search bar and press enter
6you should see a highlighted line of code
7hover over that line of code and press ctrl + B, or you can click the line to the left of the code as shown in the video
8this should make a blue arrow appear pointing at the line of code
9this is called a breakpoint, it will pause the code when we fail a dance as the fiddler crab
10press the Play button
11 before choosing an animal, copy the following:
window.send = Tp.sendBytePacket; window.squish = kr; window.token = Hc.token; window.magicNumber = 21;
12choose your animal and evolve to fiddler crab
13go onto land and purposely fail a dance, this will trigger the breakpoint
14the game will freeze, quickly navigate to the Console panel
15paste the code from step 11 and press enter
16unpause the game by pressing the blue play button ▮▶
17now copy all of the code from fiddly.js
18paste this into the Console and press enter
19now, by holding the "F" key you automatically dance :)
20enjoy the curse of immortality
